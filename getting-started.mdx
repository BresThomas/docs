## Prerequisites

- **Python 3.7+**
- **Git** (installed and configured)
- Required libraries:
  - Streamlit
  - LangChain
  - OpenAI
  - GitPython

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/BresThomas/ai-agent-blog-posts.git
cd ai-agent-blog-posts
pip install streamlit langchain openai gitpython
```

## Running the Application
Launch the app with:

```bash
streamlit run ai_blog_app.py
```
